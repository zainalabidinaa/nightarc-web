How Upcoming detection works
Movies
1
Check TMDB release_date from the detail response
2
If release_date > today → move to Upcoming with that date shown
TV Series
1
Check TMDB next_episode_to_air on the series object
2
If present → next season is announced. Show season number + air date (or "No date yet" if null)
3
If status == "Ended" → don't show in Upcoming
Refresh strategy
TMDB data is re-fetched for Liked items once per day in the background. No manual refresh needed — dates update automatically.
Inline sections

Single scrollable page — Watchlist → Liked → Upcoming stacked vertically with section headers. No tabs to switch between.

Upcoming row style

Upcoming items use a horizontal list row (not grid) so there's room to show the release badge — movie date or season info.

Auto-sort into sections

When you ❤️ a title, the app checks TMDB on save. If unreleased → goes straight to Upcoming. If available → goes to Liked. No manual sorting.

Long-press context menu

Long-pressing any item shows: Mark as Watched · Remove from Library. On Continue Watching cards: Mark Watched · Revert to previous episode · Remove.
