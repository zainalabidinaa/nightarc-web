Description sheet

Tapping anywhere on the truncated description (or "Read more →") opens a bottom sheet with the full synopsis. Episode descriptions use the same sheet when tapped in the episodes list.

Cast cards

Larger portrait cards (72×90pt) instead of small circles. Shows character role below name. Horizontally scrollable. Tapping navigates to the actor bio page.

Actor bio page

Powered by TMDB Person API. Multiple photos in a horizontal strip. Bio, personal info table, IMDb link, and "Known For" filmography row. Back nav returns to the originating show.

❤️ Like button

New heart button next to the bookmark on the detail screen — this feeds into the "Liked" tab in the new Library. Library now has two tabs: Watchlist (saved) and Liked.
