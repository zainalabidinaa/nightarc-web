Two tabs

Watchlist = bookmark (existing 🔖 button). Liked = heart (new ❤️ button on detail screen). Separate lists, both in the Library screen.

Rating badges

IMDb, Rotten Tomatoes, and TMDB score badges on each poster. Sourced from the metadata already fetched for the detail screen. Shown at poster bottom-left.

Progress bar

Purple bar at poster bottom for in-progress titles. Sourced from WatchProgressRepository — same data already used in the Continue Watching row.

Filter chips

All · Movies · Series · Recently Added. Simple client-side filter — no network calls. Persisted per tab so switching tabs remembers your last filter.
